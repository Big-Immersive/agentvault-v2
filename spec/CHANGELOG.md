# ACSS Changelog

## v1.0 — 2026-03-03

Initial release of the Agent Credential Sandboxing Standard.

- Permission Profile schema (YAML)
- Three-state access model (allow, deny, redact)
- Pattern matching (wildcard, prefix, exact)
- Last-match-wins rule evaluation
- AES-256-GCM encrypted vault format
- SQLite audit trail schema
- Session management with revocation
- Runtime metadata injection
- Conformance levels (Core, Full)
- JSON Schemas for all data types
- Example profiles (restrictive, moderate, permissive, ci-readonly, cloud-redact)
